Instructions:

1. Unzip all the code files and place them in the same directory
2. Go to terminal or any other command-line environment that supports python.
3. Within Terminal, go to the directory with all the unzipped code files.
4. Type in “python main.py”
5. Following the instructions in the terminal window.
6. The window will prompt you to input the starting location and destination in Harvard Yard.
7. The window will then prompt you to select either Dijkstra’s or Bellman-Ford’s algorithm.
8. The window then print out the shortest path according to the selected algorithm.
